Smart Garbage Monitoring System

A real-time IoT-based Smart Garbage Monitoring System using ESP32, Ultrasonic Sensor, and Gas Sensor. This project helps monitor garbage bin fill level and odor status through a live web dashboard.

Features
Real-time garbage level monitoring
Odor and harmful gas detection
ESP32-based wireless monitoring
Live web dashboard using WiFi
Automatic status updates every 3 seconds
Color indication for bin level
Gas alert system for bad odor detection
Technologies Used
ESP32
Arduino IDE
HTML/CSS
WiFi Module
Embedded C++
Components Used
ESP32 Development Board
Ultrasonic Sensor (HC-SR04)
MQ Gas Sensor
Jumper Wires
Garbage Bin Prototype
Working
The ultrasonic sensor measures the distance between garbage and the sensor.
The system calculates the bin fill percentage.
The gas sensor detects odor and harmful gases inside the bin.
ESP32 hosts a web server to display live monitoring data.
Users can access the dashboard through the ESP32 IP address.
Output
Bin fill percentage display
Distance measurement in cm
Fresh / Bad Odor / Harmful Gas status
Live visual bin indicator
